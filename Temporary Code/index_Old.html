HTML Forge

HTML Forge is a Web-Based application that converts a physical Graphical User Interface mockup into HTML5 code by leveraging the power of Machine Learning. HTML Forge was designed to speed up work-flow, and allow developers and students alike to speed up their web development tasks by simplifying the process into a simple upload and convert button. HTML Forge reduce repetitive work of a web developer, and help kickstart the project.
